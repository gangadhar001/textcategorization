This sample code package provides two functionalities. 

1: stemmer.jar. It processes a text document and produces a list of its component words and corresponding frequencies.
   To execute, run "java -jar stemmer.jar file_one.txt file_two.txt ...". The corresponding output file of word and frequency list will be named as output_file_one.txt ...
   
2: Similarity.jar. It takes the word and frequency list of 2 documents and computes the similarity between them.
   To execute, run "java -jar Similarity.jar output_file_one.txt output_file_two.txt"
